BASH CUSTOMIZATIONS

    Colors:
            
    RED="\[\e[1;31m\]";
    GREEN="\[\e[1;32m\]";
    YELLOW="\[\e[1;33m\]";
    BLUE="\[\e[1;34m\]";
    PURPLE="\[\e[1;35m\]"; 
    AQUA="\[\e[1;36m\]";
    WHITE="\[\e[1;37m\]";
    RESET="\[\e[0m\]"; 
                 
    Prompt example: PS1="${GREEN}\u${WHITE}@${RED}\h ${WHITE}in ${AQUA}\w ${WHITE}> ${RESET}";
                 
    Aliases example:
               
    alias grep='grep --color=auto'
    alias nano='nano -ic'

OPENSSL TRICKS

    Get all SSL certificates from site:

    echo | openssl s_client -showcerts -connect www.example.com:443 | awk '/BEGIN CERT/,/END CERT/ {print $0}' > ca-bundle.pem 
    echo | openssl s_client -showcerts -servername www.example.com -connect www.example.com:443 | awk '/BEGIN CERT/,/END CERT/ {print $0}' > ca-bundle.pem

    Read SSL certificate: openssl x509 -noout -text -in certificate.cer

    Extract certificate chain from PFX: openssl pkcs12 -in certificate.pfx -nokeys -out ca-bundle.pem -nodes
    Extract private key from PFX: openssl pkcs12 -in certificate.pfx -nocerts -out ca-bundle.pem -nodes 
    Remove private key passphrase: openssl rsa -in private_w_pf.key -out private.key

CURL TRICKS

    Test HTTPS: curl -vI --cacert ca-bundle.pem "https://www.example.com"
    Check certificate validity: curl -kvI https://example.com 2>&1 | awk '/Server certificate/,/issuer/ { print }'

SCREEN TRICKS

        Create named screen: screen -S screen_name
        Join named screen: screen -x screen_name
        Detach from screen: screen -d
        Reattach to screen: screen -r screen_name

        Shortcuts:

        - Detach from screen: Ctrl + a + Ctrl + d
        - Split vertically: Ctrl + a + |
        - Split horizontally: Ctrl + a + S
        - Change window: Ctrl + a + Tab
        - Start new screen session in window: Ctrl + a + Ctrl + c
        - Exit all screen windows: Ctrl + a + \
